# Computer-Vision-LAB-Exercises
The repository contains Computer Vision LAB exercises which cover the following 
- LAB-01: Image Formation
- LAB-02: Camera Calibration
- LAB-03: Harris Corner Detection, Noise & Filtering, Basic Image Processing Techniques and Edge Detection
- LAB-04: Image Pyramids
- LAB-05: Frequency analysis and Fourier Transform
- LAB-06: Template Matching
- LAB-07: Histogram of Oriented Gradients (HOG)
- LAB-08: Image Segmentation
- LAB-09: Optical Flow
- Exam  : Scale-Space Pyramid(Laplacian & Gaussian)
